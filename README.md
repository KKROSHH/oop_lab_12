# Робота з файлами та винятками
**Мета**: створити, відлагодити та протестувати програму, у якій реалізувати двонаправлений файловий ввід/вивід для обробки даних.

## Вказівки до роботи
Відповідно до цієї інструкції вам потрібно у середовищі Visual Studio Code з використанням набору компіляторів GCC створити програму мовою C++ з базовим (батьківським), похідними (дочірніми) класами для обробки даних, також реалізувати відповідні конструктори, деструктори, методи для двонаправленого файлового вводу/виводу для обробки даних. Використайте динамічний масив (вектор) структур для зберігання введених даних. Потрібно реалізувати багатофайловий проєкт у середовищі Visual Studio Code з описом кожного класу у відповідних парах заголовного та виконуваного файлів.
